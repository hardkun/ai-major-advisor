# 数据范围与免责声明

## 数据边界

当前项目不是完整高考志愿填报系统。

当前数据是部分高校公开数据验证集，用于项目演示、技术能力展示和流程验证。

项目不承诺覆盖所有高校、所有专业、所有省份、所有批次，也不承诺数据实时更新。

项目不承诺录取概率，不提供“保录取”“精准预测录取概率”等服务。

推荐结果、AI 解释和报告内容仅供信息参考，不构成最终志愿填报建议。

## 当前数据范围

- 省份：四川省为主
- 年份：2025 年为主
- 科类：物理类优先，同时兼容历史遗留的理科 / 文科写法
- 专业方向：AI、计算机、软件、电子信息、自动化、智能制造等相关专业
- 数据状态：部分已核验高校公开数据 + 部分 raw 待核验数据 + 明确标记的 demo/test 数据源

## 真实填报依据

实际志愿填报应以以下权威渠道为准：

- 省级招生考试机构
- 阳光高考平台
- 高校招生官网
- 高校招生章程
- 官方招生计划
- 本省官方志愿填报系统

## 数据治理原则

自动采集数据只进入 raw_admission_records。

人工核验通过后，数据才会进入 admissions 正式推荐表。

推荐接口只使用 admissions 中的正式数据。

测试源、示例源、本地静态源会通过 is_demo 标记或在覆盖率报告中排除，避免影响作品集演示判断。
