# AI专业择校助手｜AI相关专业推荐与招生数据采集系统

## 项目定位

AI专业择校助手是一个面向 AI 相关专业方向的择校参考系统。项目围绕公开招生数据采集、结构化处理、人工核验、推荐匹配和 AI 解释生成，构建了一个完整的业务闭环。

本项目当前定位为求职作品集项目，重点展示后端工程、数据治理、采集管线、规则推荐和大模型应用能力。当前数据集是部分高校公开数据验证集，不作为真实志愿填报依据。

## 核心用户流程

用户输入省份、分数、位次、科类和目标 AI 方向。

系统基于已核验招生数据匹配院校专业，返回冲 / 稳 / 保、往年最低分、最低位次、数据来源和核验状态。

如果启用 AI 解释，系统会基于推荐结果生成学习重点、适合人群、就业方向和风险提示。

用户可以查看免费报告，并通过模拟支付流程查看完整报告结构。

## 后台数据流程

数据源发现 → 数据源检测 → 采集预览 → HTML / Excel / PDF 解析 → raw 数据入库 → 人工核验 → 进入正式 admissions → recommend 使用正式数据。

自动采集数据不会直接进入推荐库。只有人工核验后的记录才会写入 admissions 表，推荐接口也只使用 admissions 表。

## 项目亮点

- raw/admissions 双层数据设计，区分原始采集数据和正式推荐数据。
- Human-in-the-loop 人工核验流程，避免自动采集数据直接影响推荐结果。
- 数据来源可追溯，推荐结果可展示 source_name、is_verified 等字段。
- 支持 HTML 表格、Excel、PDF 等多种公开数据结构化解析。
- 支持字段映射、表头自动识别、采集预览和采集日志。
- 推荐排序由规则算法决定，AI 只负责解释，降低高责任场景中的不可控性。
- 提供覆盖率报告、缺口诊断、候选源官方过滤和 admin 数据治理后台。
- 不是简单聊天 Demo，而是“数据工程 + 推荐系统 + AI 解释 + 管理后台”的组合项目。

## 项目边界

当前项目不是完整高考志愿填报系统，不承诺覆盖所有高校，不预测录取概率，也不构成最终志愿填报建议。正式填报应以省级招生考试机构、阳光高考平台和高校招生官网为准。
